## 2026-06-07｜文脈で覚えるIELTS英単語（Section 03-10）

### ★優先度仕分け（全技能 R/L/W/S 考慮）

> 基準：①トピック汎用性 ②アカデミック頻度 ③素直に産出できるレジスター。慣用句・文学的表現・話題限定語は低優先（認識のみ）。

#### S＝最優先（4技能すべてで効く・必ず産出できるように）
alter / transform（変える）, factor / element（要因）, deteriorate（悪化する）, abundance（豊富さ）, ideal / optimal（理想的な）, unique（独自の）, stimulate / trigger（刺激する）, uncertainty（不確かさ）, generate / produce（生み出す）, predict / anticipate（予測する）, complex / complicated（複雑な）, intense（強烈な）, perspective / viewpoint（視点）, interpret（解釈する）, notion / concept（考え方）, scepticism（懐疑）, credible / plausible（もっともらしい）, innovative（革新的な）, value / worth（価値）, disparity / gap（隔たり）, scarcity（希少性）, progress / advancement（進歩）, foster / nurture（育む）, independence / autonomy（自立）, compensate for（埋め合わせる）, creativity（創造性）, invest in（投資する）, exorbitant / excessive（法外な）, influence / persuade（左右する）, struggle（奮闘する）, prestigious / renowned（名高い）, persevere / persist（耐え忍ぶ）, emphasise / highlight（強調する）, adopt（採用する）, subsequent（その後の）, indication / sign（しるし）, sophisticated（洗練された）, manipulate（操作・説得する）, trait / characteristic（特徴）, exaggerate（誇張する）, tension / stress（緊張）, investigate（調査する）, conceal / disguise（隠す）, maturity（成熟）, obstacle / impediment（障害）

#### A＝中（読解・聴解で頻出。余裕があれば産出へ）
depart（出発する）, lack / shortage（不足）, doubt（疑い）, initially（最初は）, discovery / finding（発見）, in collaboration with（協力して）, nuisance / inconvenience（迷惑）, construction / development（建設）, surplus / excess（余剰）, encroach on（侵害する）, instinct（本能）, theory / argument（説）, obvious / self-evident（明らかな）, pattern / sequence（パターン）, surge（急増）, anxious（不安な）, automation（自動化）, area of expertise / speciality（専門分野）, be made redundant / be laid off（解雇される）, pose a problem / raise a question（疑問を投げかける）, gifted / talented（才能ある）, blueprint / framework（計画）, self-reliance（自立）, trial and error（試行錯誤）, grant / scholarship（補助金）, socially disadvantaged（恵まれない）, confidence in（自信）, rely on / depend on（頼る）, overnight success（一夜の成功）, top-notch / first-rate（一流の）, metropolitan / urban（大都市の）, groundbreaking / revolutionary（画期的な）, cut expenditure / reduce spending（支出を抑える）, appetite for（欲求）, afford（余裕がある）, the whole picture（全体像）, decelerate / slow down（減速する）, lengthy delay（長期の遅れ）, authorise / approve（認可する）, sole / lone（唯一の）, approval / support / backing（支援）, tenacity / perseverance（粘り強さ）, come up with / conceive（思いつく）, innate / natural（生まれつきの）, radical change（急進的変化）, force / compel（強制する）, advocate of / champion of（擁護者）, troubling / worrisome（問題となる）, linguistic / verbal（言語の）, standing / position（立場）, appease / placate（なだめる）, milestone / turning point（節目）, delve into（掘り下げる）, cognition（認知力）, on track（順調に）

#### B＝低（認識できればOK。産出は不要）
recover / salvage, the ocean floor / seabed, watertight / seal shut, tight timetable / strict schedule, sink / go under, remains / debris / wreckage, site / location, intact / undamaged, split in half / break in two, leave exposed, corrode / erode, treasure trove / artefacts, in hindsight, accounts vary / opinions differ, previous attempt, overpopulated / inundated, inhabit / populate, domicile / residence, route / access, grow agitated, strike / impact, trial run, relocate / transport, predator / natural enemy, landscape / terrain, make a beeline for, coordination / harmony, puzzle / mystery / riddle, mounting evidence, mirror / mimic, professional / expert, straightforward / uncomplicated, spectacular / impressive, prove impossible, from scratch, fool / dupe / mislead, lay person / the man on the street, haphazard / erratic, refuge / sanctuary, coddle / overprotect, immediate family / blood relative, know-how, spur, in one's care, like something from the pages of a novel / far-fetched, bankroll, zip / dart, place sb in harm's way, take a spin / have a go, set down / enact / legislate, sway, icon / luminary, challenge convention / buck social norms, backlash / retaliation, prodigious / exceptional, replicate / duplicate, receive recognition, far and wide, deceitful / deceptive, inflate, mask, revealing / tell-tale, sudden leap / rapid advancement
