# 実装プロンプト：英語ページの再構成（ヘッダ整理＋単語ページをテーマ選択UIに）

> English ページのIAを分かりやすくする。作り直さず統合。まず現状把握→計画→承認→実装。
> データは `web/public/content/index.json`＋テーマ別シャード（各 theme ファイルに `items` と `passage` あり）。規約は `CLAUDE.md`。

## 1. ヘッダ（English 内ナビ）を変更
- **削除**：`ライブラリ`、`会話`（会話データは現状0件。将来追加時に戻す）。
- **新ヘッダ（この6つ）**：`単語` / `文法` / `構文` / `あいまい一覧` / `類義語クイズ` / `設定`。
- 既存の「カテゴリ常設ボタン（単語/構文/文法/会話）」はこのヘッダに統合（重複を残さない）。

## 2. 単語ページを「テーマ選択UI」に全面変更（類義語クイズの構成に合わせる）
現状の単語ページ（カテゴリ＋モード＋セット量などのボタンが並ぶ画面）を、**まずテーマを選ぶ画面**に変える。**類義語クイズが既にテーマ別でよくできているので、その構成に合わせる**。

- **入口＝テーマ選択**：`index.json` の english/ielts-vocab の themes からテーマ番号ボタン（番号＋themeName＋件数）。テーマが増えたら `1–10 / 11–20` レンジ切替（`docs/Cursor_テーマ番号ボタンプロンプト.md` 準拠）。類義語クイズのテーマ選択UIと**同じ見た目・操作**に統一。
- **テーマを選ぶ→そのテーマの学習ビュー**（テーマの「長文」と「単語」を学べる）：
  - **長文（passage）**：そのテーマファイルの `passage` を表示（英文＋和訳、対象語ハイライト、文ごと🔊、linking/tips）。文脈モードの簡易版でよい（本格版は `docs/Cursor_文脈モードプロンプト.md`）。
  - **単語（words）**：そのテーマの `items`(type=word/phrase) をフラッシュカード学習。出題モード（見出し→意味／意味→見出し／例文穴埋め）、セット量(10/30/50)、並び(ランダム/昇順)、SRS採点は**このテーマ学習ビューの中**に置く（トップに散らさない）。
- つまり「テーマを選ぶ」→「長文で文脈をつかむ」→「単語をカードで覚える」の流れ。**旧『長文』タブは廃止**し、テーマ内に統合。

## 3. 各ヘッダ項目の中身
- **単語**：上記テーマ選択→テーマ学習（長文＋単語）。
- **文法**：type=grammar の項目を学習（テーマ非依存でも可。一覧＋カード）。
- **構文**：type=phrase（コロケーション・定型）を学習。必要ならテーマ選択を流用。
- **あいまい一覧**：SRSで「あいまい/苦手」と判定された語（lapses多 or 手動★）の横断一覧。タップで即学習・🔊再生。（旧『苦手だけ』フィルタを独立ページ化）
- **類義語クイズ**：既存のまま（テーマ別）。単語ページの新UIはこれに揃える。
- **設定**：同期(Worker URL/トークン)、アクセント(en-GB/US/AU)、1日の新規上限、その他。

## 4. 共有・厳守
- 既存の SRS / 発音(TTS・辞書) / 録音 / データ取得(index→遅延) を流用。重複実装しない。
- データ駆動（テーマ/件数/passage は index＋シャードから）。会話タブ削除でデータが消えるわけではない。
- 既存機能を壊さない。モバイル/a11y。新ライブラリは理由を添えて確認。

## 5. 進め方
1. 現状把握（現単語ページの構成、類義語クイズのテーマ選択実装、passage表示の有無）＋計画を提示し承認。
2. ヘッダを新6項目へ。`ライブラリ`/`会話` 撤去、重複ボタン統合。
3. 単語ページをテーマ選択UI化（類義語クイズと共通のテーマ選択コンポーネントに）。
4. テーマ学習ビュー（長文＋単語、モード/セット量/並び/SRSを内包）。
5. あいまい一覧ページ。設定ページ整理。受け入れ確認。

## 6. 受け入れ基準
1. English ヘッダが `単語/文法/構文/あいまい一覧/類義語クイズ/設定` の6項目（ライブラリ・会話なし）。
2. 単語ページは**まずテーマを選ぶUI**で、類義語クイズと同じ構成・操作感。
3. テーマを選ぶと、そのテーマの**長文(passage)と単語(カード)**を学べる（モード/セット量/並び/SRSはビュー内）。
4. あいまい一覧で苦手/あいまいな語が横断表示され、即学習・発音できる。
5. 既存（類義語クイズ・SRS・発音・録音・同期）は従来どおり。旧『長文』タブは単語のテーマ内に統合済み。
